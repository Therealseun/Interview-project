Mall of America — Interactive Sales Deck

A cinematic, browser-based sales tool designed to position the Mall of America as a high-impact platform for retail brands, sponsors, and event partners.

This project replaces traditional static presentations with an immersive, video-first experience that communicates scale, energy, and commercial opportunity within seconds.

Project Objective

The goal of this project is to build a non-linear, interactive sales experience that:

- Instantly communicates the scale and value of the property  
- Creates emotional buy-in within the first 10 seconds  
- Drives real business actions:
  - Retail leasing  
  - Sponsorship partnerships  
  - Event bookings  

This is not a traditional website. It is a digital sales tool designed for:
- Live screen-sharing presentations  
- Self-guided exploration by decision-makers

Design Approach

Visual Direction
Inspired by:
- Apple (minimal, high-impact UI)  
- Hermès (luxury tone and spacing)  
- DigiDeck (interactive presentation flow)  

---

Aesthetic System
- Dark, cinematic base (charcoal/black)  
- Gold accents for premium feel  
- Bold typography for clarity and authority  

---

Motion Philosophy
- Subtle, smooth, intentional  
- No unnecessary animations  
- Focus on storytelling, not decoration  

---

Architecture Decisions

- Built as a single-file application for simplicity and portability  
- Designed with expandability in mind:
  - Future modularization into components  
  - Potential CMS integration  
- Clean separation of:
  - Structure (HTML)  
  - Styling (CSS)  
  - Behavior (JS)
